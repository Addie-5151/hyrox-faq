# HYROX 常见问题 (FAQ)

这是一个为 HYROX 健身跑赛事参赛者提供的常见问题解答网站。

## 功能特性

- 📱 响应式设计，完美适配所有设备
- 🔍 实时搜索功能，快速找到答案
- 📚 70+ 常见问题，8 个分类
- ✨ 美观的交互式界面

## 问题分类

1. **报名相关** - 报名流程、优惠政策等
2. **变更/退款/延期** - 订单变更、退款申请等
3. **奖励/物资领取** - 完赛物资、补领流程等
4. **健康与安全** - 医疗保险、安全须知等
5. **补给与物资** - 赛事补给、物资信息等
6. **接力相关** - 接力赛规则、队伍组建等
7. **技术与成绩** - 成绩查询、证书获取等
8. **其他问题** - 停车、直播、志愿者等

## 如何使用

1. 打开 `index.html` 文件
2. 在搜索框中输入关键词快速查找
3. 点击问题卡片展开/收起答案

## 部署到 GitHub Pages

1. Fork 或克隆此仓库
2. 在 GitHub 仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为发布源
4. 访问 `https://yourusername.github.io/hyrox-faq`

## 技术栈

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
